Wildcards

  • Definition of append never uses type F. Can simplify with wildcard:
    public static <E> void append(
       ArrayList<E> a, ArrayList<? extends E> b, int count)
    {
       for (int i = 0; i < count && i < b.size(); i++)
          a.add(b.get(i));
    }
